# Galerie Video Matieres

Application front simple basee sur Vite.

La page principale affiche une grille de videos YouTube en boucle. Chaque video represente une matiere.

Quand une matiere est cochee, elle peut passer en premier plan dans un lecteur agrandi.

## Demarrage

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
